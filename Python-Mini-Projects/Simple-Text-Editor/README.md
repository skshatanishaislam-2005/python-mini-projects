# 📝 Simple Text Editor

A lightweight and user-friendly **Text Editor Application** built with **Python and Tkinter**. This desktop application allows users to create, open, edit, and save text files through an intuitive graphical interface with a built-in menu system.

---

## ✨ Features

- 📄 Create a new blank text document.
- 📂 Open and edit existing `.txt` files.
- 💾 Save text content as a `.txt` file.
- 📝 Simple text editing interface with word wrapping.
- 📋 Menu bar with File operations.
- 🔔 Confirmation message after successfully saving a file.
- 🖥️ Clean and easy-to-use desktop GUI.

---

## 🛠️ Technologies Used

- **Python**
- **Tkinter** – GUI development
- **filedialog** – Opening and saving files
- **messagebox** – Displaying user notifications

## ⚙️ How It Works

1. The application launches a Tkinter window containing a text area and menu bar.
2. The **New** option clears the current text content.
3. The **Open** option allows users to select and load existing `.txt` files.
4. The **Save** option stores the written content as a text file.
5. The **Exit** option closes the application.
