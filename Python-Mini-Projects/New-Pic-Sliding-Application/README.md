# ✨ Advanced Photo Slideshow Application

A modern and feature-rich **Photo Slideshow Desktop Application** built using **Python, Tkinter, and Pillow**. The application allows users to open image folders, view photos with smooth transition effects, control slideshow playback, and enjoy a responsive fullscreen viewing experience.

---

## ✨ Features

- 📁 Open and load images directly from a folder.
- 🖼️ Supports multiple image formats:
  - JPG / JPEG
  - PNG
  - BMP
  - GIF
  - WEBP
  - TIFF
- ▶️ Play and pause automatic slideshow.
- ⏭️ Navigate between previous and next images.
- 🎬 Multiple transition effects:
  - Fade Transition
  - Slide Transition
  - Zoom Transition
- ⏱️ Adjustable slideshow interval (1–30 seconds).
- 📊 Animated progress bar showing slideshow timing.
- 🔢 Image counter displaying current image position.
- 🖥️ Responsive window resizing.
- ⛶ Fullscreen viewing mode.
- ⌨️ Keyboard shortcuts for quick controls.
- 🧵 Background image loading using threading for a smooth user experience.
- 🎨 Modern dark-themed user interface.

---

## 🛠️ Technologies Used

- **Python**
- **Tkinter** – GUI development
- **Pillow (PIL)** – Image processing and rendering

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone <repository-link>
```

### 2. Navigate to the Project Directory

```bash
cd Advanced-Photo-Slideshow
```

### 3. Install Required Dependencies

```bash
pip install pillow
```

### 4. Run the Application

```bash
python slideshow.py
```

---

## ⚙️ How It Works

1. Users select an image folder through the file explorer.
2. The application scans and loads supported images.
3. Images are processed using Pillow for optimal display.
4. The slideshow can be controlled using buttons or keyboard shortcuts.
5. Transition animations create smooth movement between images.
6. The application automatically adjusts images based on the window size.
