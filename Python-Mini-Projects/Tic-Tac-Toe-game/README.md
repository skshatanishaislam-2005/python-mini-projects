# 🎮 Tic-Tac-Toe Game in Python

A simple Tic-Tac-Toe game built using Python and Tkinter. This project allows two players to play against each other on a graphical 3×3 board with automatic winner detection and visual feedback.

## 🚀 Features

* Interactive GUI built with Tkinter
* Two-player gameplay (X and O)
* Automatic winner detection
* Winning cells highlighted in green
* Non-winning played cells highlighted in red
* Winner announcement using a pop-up message box
* Turn indicator showing the current player
* Clean and beginner-friendly code structure

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI Library)

## 📸 How It Works

1. Player X starts the game.
2. Players take turns clicking cells.
3. The game checks all possible winning combinations after each move.
4. When a player wins:

   * Winning cells turn green.
   * Other played cells turn red.
   * A popup announces the winner.
5. The game closes after displaying the result.
