# Rent Calculator🏠💰

A easy simple mini project that calculates rent, food expenses, and electricity costs shared among roommates.

## Features

- Calculate total rent cost
- Add food expenses
- Calculate electricity bill with per unit charges
- Split expenses among multiple people

## Technologies Used

- Python

## How It Works

The user enters:
- Monthly rent
- Number of months
- Number of people sharing the rent
- Monthly food expenses
- Electricity units consumed
- Charge per electricity unit

The program calculates the total expense and displays how much each person should pay.
