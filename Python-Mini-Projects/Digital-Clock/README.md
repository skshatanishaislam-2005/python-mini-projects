# ⏰ Digital Clock in Python

A simple and elegant Digital Clock built using Python and Tkinter. The application displays the current time and date in real-time and automatically updates every second.

## 🚀 Features

* Real-time digital clock
* Displays current hours, minutes, and seconds
* Shows AM/PM format
* Displays the current date
* Automatic refresh every second
* Simple and beginner-friendly GUI
* Built entirely with Python and Tkinter

## 🛠️ Technologies Used

* Python 3
* Tkinter
* Time Module (`strftime`)

## 📸 Preview

The clock displays:

```text
10:45:32 PM
06/12/26
```

and updates automatically every second.

## ▶️ Running the Project

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project directory:

```bash
cd digital-clock
```

3. Run the application:

```bash
python digital_clock.py
```

## 📖 How It Works

* The `strftime()` function retrieves the current system time and date.
* The `time()` function updates the label every second using Tkinter's `after()` method.
* The clock display is refreshed continuously without freezing the GUI.
