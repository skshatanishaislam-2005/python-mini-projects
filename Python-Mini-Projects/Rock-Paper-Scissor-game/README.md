# ✊📄✂️ Rock Paper Scissors Game in Python

A simple command-line Rock Paper Scissors game built using Python. The player competes against the computer, which makes random choices using Python's built-in `random` module.

## 🚀 Features

* User vs Computer gameplay
* Random computer moves
* Win, Lose, and Tie detection
* Interactive command-line interface
* Beginner-friendly Python project
* Clear result messages for every outcome

## 🛠️ Technologies Used

* Python 3
* Random Module

## 🎮 How the Game Works

1. The player enters one of the following choices:

   * Rock
   * Paper
   * Scissor

2. The computer randomly selects its move.

3. The program compares both choices and determines:

   * Winner
   * Loser
   * Tie

4. The result is displayed in the terminal.
