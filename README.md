# 📝 Task Management App (Python)

## 📌 Project Overview

The **Task Management App** is a simple command-line application built using Python. It allows users to create and manage their daily tasks efficiently through an interactive menu.

This project is ideal for Python beginners who want to practice working with:

* Functions
* Lists
* Loops
* Conditional Statements
* User Input

---

## ✨ Features

* ➕ Add new tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* 👀 View all tasks
* 🚪 Exit the application

---

## 🛠️ Technologies Used

* Python 3

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Clone or download this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Open the terminal.
5. Run the program:

```bash
python Task-app.py
```

or

```bash
python3 Task-app.py
```

---

## 📖 How It Works

1. Enter the number of tasks you want to create.
2. Enter each task name.
3. Select an option from the menu:

```
1 - Add Task
2 - Update Task
3 - Delete Task
4 - View Tasks
5 - Exit
```

4. Continue managing your tasks until you choose **Exit**.
